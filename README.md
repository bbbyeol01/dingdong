# Dingdong
[개발자 커뮤니티]<br>
개발이라는 하나의 주제 안에서, 타 커뮤니티보다 조금 더 가볍고 자유로운 이야기를 나눌 수 있는 사이트를 만들고자 했습니다. 게시글에서 만난 유저들의 인연이 편하게 이어지길 바라는 마음에서 스터디룸 예약 기능을 추가했습니다.
